# ScatVer

Versioning for the erratic hobbyist dev.

```py
╭╴Era    ╭╴Drop
0.202505.1FB
  ╰╴YearMonth
```

---

## Indicators

### Era

```ts
P ─ Prototype
0 ─ Era
```

### Drop

```ts
1 ─ Drop Count [This Month]

/// Change Types
F ─ Feature
B ─ Bug Fix
S ─ Security Fix
I ─ Incompatible (Breaking)
E ─ Experimental Feature
D ─ Dependency Bump
```
